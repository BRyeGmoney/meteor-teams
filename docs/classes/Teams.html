subscription

Object

Defined in client/subscriptions.js:4

Subscription handle for the currently logged in user's permissions.

NOTE: The corresponding publish function, _teams, depends on this.userId so it will automatically re-run when the currently logged-in user changes.

Example:

Teams.subscription.ready() // => true if user teams have been loaded
